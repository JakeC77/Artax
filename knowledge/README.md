## Supported Formats
- Text: `.txt`, `.md`, `.json`, `.yaml`, `.csv`

- Images: `.jpg`, `.png`, `.gif`, `.webp`

```

  clients/         # Client info
  research/        # Articles, papers
  reference/       # Quick reference material
```

Just commit files here. Artax will read them when relevant.
